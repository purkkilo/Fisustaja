# FisuJS
Fishing competition application

# Run The application server

## Server setup, Run backend server, dev version with nodemon
```
npm install
npm run dev
```

### Client setup and init
```
Open new console
cd client
npm install
npm run serve

For more see: FisuJS/client/README.md 
```
